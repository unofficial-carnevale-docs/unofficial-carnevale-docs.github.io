This site is a community-maintained reference for *Carnevale*, the tabletop skirmish game published by [**TTCombat**](https://ttcombat.com/), set in a romanticized alternate history Venice of 1795, during the height of the Carnival season.

Content covers game mechanics analysis, faction profiles, magic school breakdowns, model profiles, and strategic guides. All game content and intellectual property belongs to TTCombat — this site provides fan-made analysis and reference material only. It was created using freely accessible [**Carnavale ressources**](https://ttcombat.com/collections/carnevale).
